# Git Guideline

## Branch Naming

Use this pattern for branch names:

```text
(type)/{GITHUB_ISSUE_ID}
```

- `type`: `feature`, `fix`, `enhance`

## Branch Lifecycle

1. Branch from `master`.
2. Keep commit names short and descriptive.
